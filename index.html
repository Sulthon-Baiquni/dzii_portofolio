Tentang Saya

Sulthon Baiquni Dzinni-Dhzomi

Saya memiliki keahlian yang mencakup berbagai bidang pada pengembangan Web, Mobile, 3D desain, dan Game. Untuk pengembangan web saya menguasai HTML, Tailwind CSS, Laravel, dan Flask. Project Yang Telah Dibuat

Berikut merupakan beberapa contoh project yang pernah dibuat mulai dari pengembangan desain UI / UX, Website, 3D Desain, Mobile App dan Game.

Flutter Mobile App - E-commerce API Strapi

Sebuah mobile app yang berisikan mengenai e commerce dimana pada tampilannya sudah menggunakan implementasi API. API yang digunakan dari strapi, pada tampilannya memiliki empat menu yaitu home, menu, kategori dan akun. Selain impementasi API, mobile app ini dilengkapi bloc state management untuk mengelola akses register dan login.

Sayangnya Aplikasi ini Tidak Dihosting

2D Game Platformer

Sebuah game dengan gaya seni dua dimensi bertipe platformer dengn memiliki mekanik untuk berjalan, meloncat dan menghindar musuh. Dilengkapi dengan sistem coint dan live health, serta animasi di setiap gerakan. Selain itu dilengkapi juga dengan User Interface yang friendly dengan memiliki dua scene menu seperti start, quit, pause dan resume.

Sayangnya Game ini Tidak dapat dimainkan secara online

Web - SPK Konsol Game

Sebuah website yang dibuat dengan bantuan framework laravel dan tailwind CSS (flowbite plugin). Pada website user dapat melalukan pemilihan konsol game terbaik berdasarkan input nilai bobot dari beberapa kriteria yang telah disediakan. Selain itu website ini juga dilengkapi dengan tampilan admin + CRUD dan metode yang diSPK yang digunakan adalah Weighted Product.

Sistem Pendukung Keputusan Konsol Game Terbaik

Web - Tourism

Sebuah website sedarhana dengan tema Travel and Tourism yang dibuat dengan html, bootstrap dan juga javascript. Pada website ini memiliki beberapa seperti home, fitur, location, service, dan blog. Website sudah dimenerapkan responsive design, dimana ketika diakses pada beberapa perangkat seperti pc maupun smartphone layoutnya akan tetap rapi.

Sayangnya Website ini Tidak Dihosting
